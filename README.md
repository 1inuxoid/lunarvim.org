# LunarVim.org

Website for [LunarVim](https://www.lunarvim.org/)

j

## Run this site locally

- Clone the repo:

```
git clone https://github.com/LunarVim/lunarvim.org.git
```

- Install:

```
cd lunarvim.org && npm install
```

- Start the server:

```
npm run docs:dev
```

- Run the site in your browser:
 
```
http://localhost:8080/
```
